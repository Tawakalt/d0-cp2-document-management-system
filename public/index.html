    Introduction

    Welcome to the Document Management System API! You can use our API to access Document Management System API endpoints, which allows you manage your documents and have access to others in our database. We have language bindings only in Javascript! You can view code examples in the dark area to the right.

    Roles

    Create Roles

    Request:

    {
  "role": "Editor"
}

    Response:

    {
    "message": "Role successfully created"
}

    Request

    • Endpoint: POST: /api/v1/roles/
    • Body (application/json)

    Response

    • Status: 201: Created
    • Body (application/json)

    Errors

    This error section is stored in a separate file in `includes/_errors.md`. Slate allows you to optionally separate out your docs into many files...just save them to the `includes` folder and add them to the top of your `index.md`'s frontmatter. Files are included in the order listed.

    The Document Management System API uses the following error codes:

    Error Code Meaning Description
    200 OK Successful Request
    201 Created Post Request Successful
    400 Bad Request Your request is faulty.
    401 Unauthorized You are unauthorized to make the request
    404 Not Found The specified user/document could not be found.
